# T5 - Coloracao de Grafos com DSatur

Implementacao em **Python** do Trabalho Pratico 5 da disciplina
**Resolucao de Problemas com Grafos**.

## Requisitos

- Python 3.6 ou superior (sem dependencias externas)

## Estrutura

```text
T5/
├── README.md
├── ROTEIRO.md
├── dados/
│   ├── brasil.txt
│   ├── teste-caminho4.txt
│   ├── teste-ciclo4.txt
│   ├── teste-ciclo5.txt
│   └── teste-triangulo.txt
└── src/
    ├── graph.py
    ├── dsatur.py
    └── main.py
```

## Execucao

Execute os comandos **a partir da raiz do projeto** (pasta `T5/`).

### Entrada oficial

```bash
python3 src/main.py dados/brasil.txt
```

### Fixtures de teste

```bash
python3 src/main.py dados/teste-triangulo.txt
python3 src/main.py dados/teste-caminho4.txt
python3 src/main.py dados/teste-ciclo4.txt
python3 src/main.py dados/teste-ciclo5.txt
```

> **Windows:** substitua `python3` por `python` caso necessario.
