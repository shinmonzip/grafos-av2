# t2-cavalo

## Requisitos

- Python 3.10+

## Como executar

1. Abra um terminal na raiz do projeto (t2-cavalo).
2. Execute um dos comandos abaixo:

```bash
python3 src/main.py
```

```powershell
py src/main.py
```

Opcionalmente, voce pode informar outro arquivo de entrada:

```bash
python3 src/main.py --input dados/entrada.txt
```

```powershell
py src/main.py --input dados/entrada.txt
```

## Arquivo de entrada

O programa le o arquivo:

- dados/entrada.txt

Formato esperado (algs4):

1. Linha 1: numero de vertices V
2. Linha 2: numero de arestas E
3. Proximas E linhas: pares v w
