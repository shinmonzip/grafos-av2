# Trabalho Pratico 3 - Grafos (T3)

## Requisitos

- Python 3.9+ instalado no sistema

## Execucao

Na raiz do projeto, rode:

```bash
python3 src/main.py
```

Para informar outro arquivo de entrada:

```bash
python3 src/main.py dados/entrada_original.txt
python3 src/main.py dados/entrada_eulerizada.txt
```

## Formato de entrada

```text
V
E
v w peso
v w peso
...
```

## Saida esperada para o exemplo eulerizado

- Tabela de graus de entrada e saida por vertice
- Circuito euleriano
- Custo total igual a 10
